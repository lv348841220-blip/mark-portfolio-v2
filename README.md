# Mark 吕连生 | 短视频内容运营 & IP孵化

个人作品集网站。

## 部署

### GitHub Pages（推荐，最简单）

1. 创建 GitHub 仓库 `mark-portfolio`
2. 推送代码
3. Settings → Pages → Source: main branch
4. 访问 `https://你的用户名.github.io/mark-portfolio/`

### Vercel

1. 连接 GitHub 仓库
2. 自动部署，零配置
3. 获得 `xxx.vercel.app` 域名

### Cloudflare Pages

1. 连接 GitHub 仓库
2. Build command: 留空（纯静态）
3. Output directory: `/`
